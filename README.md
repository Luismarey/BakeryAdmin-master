# BakeryAdmin - Starter project

This is a minimal ASP.NET Core 8 MVC starter project with EF Core (LocalDB) and Identity (roles seeded).

## Run

- Requires .NET 8 SDK
- From project folder:

```
dotnet restore
dotnet build
dotnet run
```

The project will create a LocalDB database named `BakeryAdminDb` automatically.

Default seeded admin:
- email: admin@bakery.local
- password: Admin#1234
